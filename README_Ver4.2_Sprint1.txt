地域限定旅行業務取扱管理者 Ver4.2 Sprint1

【今回追加した機能】
1. 全25問の「続きから」
   ・途中でメニューへ戻っても進行位置を端末内に保存します。
   ・次回、ホーム画面の「続きから」で再開できます。

2. 「リセット」
   ・ホーム画面または全25問の問題画面から実行できます。
   ・確認画面で「OK」を押した場合のみ、途中経過を消して第1問から始めます。
   ・正誤履歴、間違えた問題、自分用メモは消しません。
   ・消えるのは全25問の「現在位置」だけです。

【GitHub更新方法】
ZIPを解凍し、中のファイルをGitHubリポジトリ直下へ上書きしてください。
index.html / questions.js / categories.js / manifest.webmanifest / sw.js をすべて更新します。

【古い画面が出る場合】
SafariまたはChromeでページを再読み込みしてください。
ホーム画面アイコンから古い版が開く場合は、一度アイコンを削除し、
Safariで公開ページを開いて「ホーム画面に追加」をやり直してください。
