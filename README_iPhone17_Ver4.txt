地域限定旅行業務取扱管理者 合格専用 Ver4 iPhone17版

【Ver4で追加した機能】
1. 問題文と選択肢の読み上げ
2. 旅行業法を9分野に分けた分野別学習
3. 答え合わせ後、最初に「正解・あなたの回答・正しい解答」を表示
4. ランダム5問セットと5問終了後の結果表示

【GitHubへの更新】

index.html
questions.js
categories.js
manifest.webmanifest
sw.js

既存の同名ファイルは上書きしてください。

【読み上げについて】
iPhone標準の日本語音声を利用します。
最初の読み上げ開始時に少し間が空くことがあります。
「読み上げ停止」で途中停止できます。
